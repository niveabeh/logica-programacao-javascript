estrutura de repetição

As estruturas de repetição em JavaScript são essenciais para executar um bloco de código várias vezes automaticamente, facilitando tarefas repetitivas e deixando o código mais eficiente. As principais são o for: usada quando se sabe o número exato de repetições combinando inicialização, condição e incremento em uma linha; o while, que repete o código enquanto uma condição for verdadeira, ideal quando não se sabe quantas vezes será necessário executar; e o do...while, similar ao while, mas garante que o código seja executado ao menos uma vez antes de checar a condição. Essas estruturas ajudam a criar programas mais dinâmicos, organizados e fáceis de manter.

for

A estrutura for é uma das mais utilizadas. Ela permite executar uma sequência de comandos um número determinado de vezes. É muito útil quando o número de repetições é conhecido de antemão, como ao percorrer uma lista de elementos ou realizar cálculos iterativos. Seu formato compacto combina início, condição de parada e incremento em uma única linha, o que facilita o controle do loop.

while

A estrutura while executa um bloco de código enquanto uma condição for verdadeira. Ao contrário do for, o while é ideal para situações em que não se sabe quantas vezes o código precisa ser repetido, mas se sabe a condição que determina a continuação da execução. Ele é especialmente útil em cenários de espera, verificações contínuas ou interações com o usuário.

do while

O do...while é semelhante ao while, mas com uma diferença importante: ele garante que o bloco de código será executado pelo menos uma vez, mesmo que a condição seja falsa desde o início. Isso acontece porque a verificação da condição ocorre após a execução do bloco. Essa estrutura é útil quando é necessário executar uma ação inicial antes de verificar se deve continuar repetindo.
