O algoritmo lê três números inteiros

O algoritmo lê três números inteiros informados pelo usuário e determina qual deles é o maior. Ele utiliza a estrutura de repetição while para garantir que exatamente três números sejam lidos. A cada entrada, verifica se o número informado é maior que o atual maior valor registrado e, ao final, exibe o maior número encontrado.
